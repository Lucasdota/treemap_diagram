## Treemap Diagram

### A treemap diagram with JSON data fetching from 3 different URLs(video games, movies and kickstartes) using D3 library to visualize data.

Main features:

- Data fetch
- Manipulating JSON
- Determine sizes according to key:value of each array in the object
- Hierarchy() method
